## Overview

### Event Statistics

- Average item price

- Commission totals
- Fees collected

- Total consignors
- Type breakdown (Regular/Super)
- Volunteer shifts filled
- Drop-off schedule status

Financial:

- Fee waivers
- Earnings distribution
- Return impact
